# Terra Indomita Landing Page

A single-page website for Terra Indomita, an outdoor education school. The layout mirrors the provided mockups and highlights the organisation, its services, founder, and contact information.

## Page Sections
- Hero banner with crest, key service list, and forest background.
- "Qui sommes-nous" presentation with crest and mission statement.
- Service trio detailing training camps, immersive camps, and private mentoring.
- Founder spotlight with bio and portrait.
- Contact block linking to social platforms and email.

## Project Structure
```
index.html              # Page markup
assets/css/styles.css   # Styling and responsive rules
assets/images/          # Logo, photography, and social icons
```

## Usage
1. Replace the placeholder assets inside `assets/images/` with your final imagery, keeping the same filenames or updating the `<img>` paths in `index.html`.
2. Open `index.html` directly in a browser (`file:///.../index.html`) or serve the folder locally (e.g., `python3 -m http.server`).
3. Update copy, links, or colours as needed in `index.html` and `styles.css`.
